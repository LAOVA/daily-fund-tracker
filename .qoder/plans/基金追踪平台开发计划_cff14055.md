# 基金追踪平台开发计划

## 技术栈

- **框架**: Next.js 14+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS + shadcn/ui
- **图表**: chart.js + chartjs-finance
- **状态管理**: Zustand + localStorage 持久化

## 实现步骤

### 步骤 1: 初始化 Next.js 项目

```bash
npx create-next-app@latest demo --typescript --tailwind --eslint --app --src-dir=false --import-alias="@/*" --use-npm --no-turbo
```

### 步骤 2: 安装依赖

```bash
# shadcn/ui 初始化
npx shadcn-ui@latest init

# 安装 shadcn/ui 组件
npx shadcn-ui@latest add button card input table tabs dialog toast select slider switch dropdown-menu badge scroll-area skeleton

# 安装图表库
npm install chart.js chartjs-finance react-chartjs-2

# 安装状态管理
npm install zustand

# 安装其他依赖
npm install lucide-react axios
```

### 步骤 3: 配置 Tailwind 和样式

- 配置 Tailwind 使用 fund-tracker-style-guide.md 中的配色方案
- 创建全局样式变量

### 步骤 4: 创建后端 API 代理

**API 路由**:

- `/api/funds/valuation` - 获取基金估值数据
- `/api/funds/holdings` - 获取基金持仓数据
- `/api/funds/search` - 搜索基金

**代理逻辑**:

- 使用 axios 请求天天基金/东方财富接口
- 处理并返回 JSON 数据

### 步骤 5: 开发核心组件

1. **FundSearch** - 基金搜索输入组件
2. **ValuationTable** - 估值表格组件
3. **FundCard** - 基金卡片组件
4. **StockHoldingCard** - 重仓股票卡片
5. **FundChart** - K 线/净值图表

### 步骤 6: 开发 Zustand Store

1. **useFundsStore** - 管理自选基金列表（含 localStorage 持久化）
2. **useSettingsStore** - 管理刷新频率设置（含 localStorage 持久化）
3. **usePortfolioStore** - 管理基金分组和组合视图

###步骤 7: 开发页面

1. **首页** - 综合仪表盘
2. **基金估值页** - 基金净值查询
3. **重仓股页** - 持仓追踪
4. **组合管理页** - 自选基金

### 步骤 8: 响应式适配

- 移动端适配
- 触摸友好
