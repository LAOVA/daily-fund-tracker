# 基金追踪工具功能改进计划

## 一、数据展示完善（优先级：高）

| 功能 | 说明 |
|------|------|
| 基金详情页 | 基金类型、基金经理、规模、成立日期、费率等基础信息 |
| 净值走势图增强 | 支持选择时间段（近1月/3月/6月/1年/全部） |
| 历史净值表格 | 详细的历史净值记录，支持分页 |

## 二、持仓管理增强（优先级：高）

| 功能 | 说明 |
|------|------|
| 交易记录 | 记录买入/卖出/分红，计算真实成本 |
| 定投计算 | 定投收益模拟、定投计划管理 |
| 分红记录 | 基金分红追踪、红利再投计算 |
| 多次买入成本计算 | 支持分批建仓，计算加权平均成本 |

## 三、用户体验优化（优先级：中）

| 功能 | 说明 |
|------|------|
| 深色模式 | 实现明暗主题切换 |
| 数据导出 | 导出持仓/收益报告为 Excel/CSV |
| 数据导入 | 批量导入基金列表 |
| 基金备注 | 给基金添加个人备注 |
| 价格提醒 | 涨跌幅超过阈值时提醒 |

## 四、分析功能（优先级：中）

| 功能 | 说明 |
|------|------|
| 组合分析 | 持仓分布饼图、行业分布、基金类型分布 |
| 收益统计 | 按日/周/月/年统计收益 |
| 收益归因 | 分析哪只基金贡献了最多收益 |
| 风险指标 | 最大回撤、波动率等 |

## 五、代码优化

| 项目 | 说明 |
|------|------|
| `utils.ts` 颜色变量 | `getChangeColor` 函数改为使用 CSS 变量 |
| 移动端优化 | 小屏幕表格体验优化 |
| 错误处理 | API 失败时更友好的提示和重试机制 |

## 六、推荐优先实现顺序

1. **交易记录功能** - 让持仓管理更实用
2. **深色模式** - 用户体验提升
3. **组合分析图表** - 可视化数据更直观
4. **数据导出** - 实用性强，实现简单
5. **基金详情页** - 信息更完整
