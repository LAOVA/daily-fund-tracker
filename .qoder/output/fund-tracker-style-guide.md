# 基金估值与重仓追踪 - UIUX 风格规范

## 项目概述

- **项目名称**: 实时基金估值与重仓追踪平台
- **项目类型**: 金融数据展示 / 新闻资讯类网页
- **设计风格**: 纯白背景、报纸风格、杂志风格、现代简洁
- **技术栈**: HTML + Tailwind CSS (CDN)

## 4. 组件规范

### 卡片组件

- **背景**: `#FFFFFF` (白色)
- **边框**: 3px solid `#C9C2B5`
- **圆角**: 0px (直角，报纸风格)
- **内边距**: 20px - 32px
- **阴影**: 轻微 `hover:shadow-md`
- **悬停效果**: 边框变深，阴影出现

### 表格组件

- **表头背景**: `#F5F0E6` (强调米色)
- **表头文字**: `#2D2A26`，字重 700，大写
- **奇数行**: `#FFFFFF`
- **偶数行**: `#FAFAFA`
- **行高**: 56px (更大气)
- **边框**: 2px solid `#C9C2B5`

### 标题下划线风格

- **headline-underline**: 4px solid `#2D2A26`
- **padding-bottom**: 8px
- 模拟报纸大标题风格

### 刊头元素

- **日期期数**: 小字号，辅助灰色
- **分隔线**: 报纸边框色
- **大写字母**: 辅助字体，tracking-wider

### 涨跌幅指示器

- **上涨**: 红色 `#C41E3A`，带 ↑ 图标
- **下跌**: 绿色 `#228B22`，带 ↓ 图标
- **持平**: 蓝色 `#1565C0`，带 → 图标
- **字重**: Bold 更醒目

---

## 5. 视觉风格细节

### 报纸/杂志风格元素

1. **粗边框**: 3px 边框强调区块边界
2. **双线分隔**: 2px 底部边框区分区块标题
3. **标题下划线**: 4px 粗线模拟报纸大标题
4. **首字母大写**: Drop Cap 首字母放大
5. **刊头设计**: 日期、期号、天气信息
6. **大写标签**: 使用小字号大写字母 (kicker)
7. **左侧边框指示**: 悬停时显示彩色左边框
8. **卡片阴影**: 微妙的 hover 阴影效果

### 动效规范

- **过渡时间**: 150ms - 300ms
- **过渡函数**: ease-in-out
- **悬停效果**: 阴影出现、边框指示器
- **减少动画**: 遵循 prefers-reduced-motion

---

## 6. 交互规范

### 悬停状态

- 卡片: 阴影出现 (shadow-md)
- 按钮: 背景色变深
- 链接: 颜色变化
- 快讯: 左侧彩色边框指示器出现

### 焦点状态

- 轮廓: 2px solid `#8B0000`
- 偏移: 2px
- 确保键盘可访问

---

## 7. 可访问性要求

- [ ] 颜色对比度: 文字与背景至少 4.5:1
- [ ] 焦点可见: 所有交互元素有清晰焦点指示
- [ ] 键盘导航: 支持 Tab 键导航
- [ ] ARIA 标签: 图表和数据更新使用 aria-live
- [ ] 字体大小: 正文最小 16px
- [ ] 点击区域: 移动端最小 44x44px
